This repository contains the codes to optimise a stochastic model aiming to generate realistic GRB light curves from different GRB experiments using a genetic algorithm (GA). The model is based on a stochastic differential equation (SDE) involving a generalised geometric Brownian motion (gGBM). The GA is implemented in python via the pygad package which is needed in your environment to optimise the model. The GRB light curves can be simply generated.
